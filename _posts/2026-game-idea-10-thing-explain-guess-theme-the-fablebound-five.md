## Thing explain guess

One Liner: Can you describe complexity with simple words?

A variation of the ['up goer five'](https://xkcd.com/1133/)

Web game.

The player is shown the name/title of a complex object (Nuclear power plant, hydro electric dam, human circulatory system).

The player needs to use the top 10,000 frequently (maybe 500 words to keep with the theme) used words to describe the complex object. The goal is to use the least amount of words possible and have the fewest wrong guesses.

When the user is entering in the description, the words that are too complex are underlined in light blue, like a spelling mistake. See this page for an example <https://splasho.com/upgoer5/>

The user submits the description and it's added to the pool of descriptions.

The user can also try to guess the complex object from a series of previously submitted descriptions.

The user gets points for submitting descriptions with the lowest letter count, and shortest amount of time guessed. Or they can get points from guessing the object from other people's descriptions.

### Notes

- Each description has a max amount of guesses before it's deemed unguessable and is removed from the general pool.
- Maybe only allow people to make one guess on each description so they can't just try all the things
- The guesser and the writer of descriptions should get points
